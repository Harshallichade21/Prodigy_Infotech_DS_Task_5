## Traffic Accident Data Analysis

### 📌 Objective
Analyze traffic accident data to identify patterns related to road conditions, weather, time of day, and other contributing factors. Visualize accident trends and build a machine learning model to predict accident severity.

## 📂 Dataset
- **Dataset:** <a href="https://github.com/Harshallichade21/Prodigy_Infotech_DS_Task_4/blob/main/twitter_training.csv](https://github.com/Harshallichade21/Prodigy_Infotech_DS_Task_5/blob/main/RTA%20Dataset.csv"> Road Traffic Accident Dataset  </a>
- **Source:** Kaggle

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- imbalanced-learn (SMOTE)

## 📊 Project Workflow

### 1. Data Loading
- Imported the dataset into Python using Pandas.

### 2. Data Exploration
- Checked dataset dimensions.
- Viewed column information.
- Generated descriptive statistics.
- Identified missing and duplicate values.

### 3. Data Preprocessing
- Handled missing values.
- Removed duplicate records.
- Encoded categorical variables.
- Applied One-Hot Encoding and Label Encoding.
- Balanced the dataset using SMOTE.

### 4. Exploratory Data Analysis (EDA)
- Visualized accident severity distribution.
- Created scatter plots.
- Generated a correlation heatmap.
- Explored relationships among different variables.

### 5. Machine Learning
- Split the dataset into training and testing sets.
- Built a K-Nearest Neighbors (KNN) classifier.
- Trained the model on the processed dataset.

### 6. Model Evaluation
The model was evaluated using:
- Accuracy Score
- Confusion Matrix
- Classification Report

## 📈 Visualizations
- Accident Severity Distribution
- Scatter Plot
- Correlation Heatmap
- Confusion Matrix

## 🎯 Conclusion
This project analyzes traffic accident data to understand factors affecting accident severity. Data preprocessing, visualization, and machine learning techniques were applied to gain insights and build a predictive model for accident severity classification.
